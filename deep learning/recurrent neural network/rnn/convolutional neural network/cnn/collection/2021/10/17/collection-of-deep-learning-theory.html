📌 深度學習的理論目錄 - Collection of Deep Learning Theory

Oct 17, 2021 • 1 min read

Deep Learning   Recurrent Neural Network   RNN   Convolutional Neural Network   CNN   Collection

Prerequisite

建議看完 機器學習的理論 並學會機器學習的基礎概念後，再來研究深度學習。

Introduction

深度學習是由機器學習分支出來的一個大領域，其概念是應用神經網路為基礎，並以現今硬體 (特別是 GPU) 效能為輔助，將神經網路建立的越來越大 (深度兩字的由來)。網路內的所有神經元 (neuron) 可以自動學習到問題所需的參數，並以此來預測、解決未知的問題。

吳恩達教授的深度學習 1 是我最推薦的深度學習基礎課程之一，這裡一樣是在吸收完教學後，所寫的一些教學筆記；我將所有筆記統整在這裡，方便大家能夠一起學習。

這是新的部落格，目前還在搬運筆記中，舊筆記在: https://sejkai.gitbook.io/academic/deep-learning
